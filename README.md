# SDL3_3D
<h1><strong>PROJET D'EMPLEMENTATION</strong></h1>
Notre projet nous permet de montrer un programme ecrit en C++ en utilisant SDL3 pour afficher selon la demande un carre bleu sur un fond noir
que l'utilisateur peut deplacer de la gauche vers la droite, du haut vers le bas.

<h2>CE QU'IL FAUT POUR CONCEVOIR CE PROGRAMME</h2>
En plus d'un ordinateur, il faut un compilateur C++
(g++, clang++, et plein d'autre)
et il faut aussi avoir au prealable installe SDL3
